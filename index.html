💸 GitMoni

GitMoni is a simple social currency system built on top of GitHub commits. Users earn and send moni based on transaction JSONs stored in a GitHub repo.

🚰 Faucet

The GitMoni faucet is seeded with moni and can reward users based on GitHub activity, rank, or manual claims. A sample faucet transaction:

{
  "txid": "2025-09-05T05:42:55Z-gh:system->gh:gitmoni-crypto-1618033989",
  "from": "gh:system",
  "to": "gh:gitmoni-crypto",
  "amount": 1618033989,
  "nonce": 1,
  "ts": 1757050975467,
  "memo": "star:rank=1"
}

🌱 Philosophy

GitMoni encourages open source contributions by letting users tip, earn, and exchange a mock currency via Git. All balances and history are public, trackable, and verified through commit hashes.
