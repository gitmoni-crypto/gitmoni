What is MONI?

MONI is a community currency distributed through GitHub stars ‚≠ê. Each GitHub account can star the repository once, and receives a one-time faucet grant of MONI.

How distribution works

  • The 1st stargazer receives ~1.6 billion MONI.
  • The n-th stargazer receives g1 / n, so #10 gets 1/10th of the first grant, #100 gets 1/100th, and so on.
  • This ‚Äúharmonic decay‚Äù rewards early adopters while capping total free supply at ~40 billion MONI even if billions of accounts starred.
  • After the faucet grant, users can only earn MONI by receiving transfers in the ledger.

The Ledger

All transactions are public JSON files in the transactions branch. The Explorer provides indexes (balances.json, txmeta.json) for quick lookups.

Every transaction is validated automatically via GitHub Actions. The from field must match the GitHub user who opened the PR. This ensures identities are tied to GitHub accounts.

Get Started

  1. ‚≠ê Star the GitMoni repo.
  2. Wait for the faucet workflow (~10 min) to create your MONI grant.
  3. Check your balance in the Explorer.
  4. Submit new transfers via Submit Transaction.
