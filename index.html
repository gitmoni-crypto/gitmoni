What is MONI?

MONI is a community currency distributed through GitHub stars ‚≠ê. Starring the GitMoni repo earns a one-time faucet grant of MONI, with earlier stargazers receiving more.

How distribution works

  • The first stargazer receives exactly 1_618_033_989 MONI (the golden ratio √ó 1 billion).
  • The n-th stargazer receives 1_618_033_989 / n.
  • This harmonic decay rewards early adopters while naturally capping total supply.
  • MONI is not a real cryptocurrency ‚Äî it's an experiment in social distribution and transparent tracking on GitHub.

The Ledger

Every MONI transaction is stored as a JSON file in the transactions branch of the GitHub repo.

GitHub Actions validate each pull request. The sender (from) must match the GitHub user who submitted it. This links identity and wallet.

The Explorer builds indexes like balances.json and txmeta.json for fast lookups.

  3. Check your balance in the Explorer.
  4. Send a transaction by submitting a PR to the repo or using Submit Transaction.
