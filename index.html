What is GitMoni?

Git-backed, crypto-inspired ledger with browser-side verification. No servers, no DB ‚Äî just static files on GitHub Pages.

  • Wallet: generate keys & sign in-browser.
  • Validation: verify inputs/outputs and signatures client-side.
  • Indexes: O(1) lookups via index.json (+ optional date shards).

How to Publish New TXs

  1. Add a TX file: data/txs/<txid>.json
  2. Run locally: npm run build:index (and optionally npm run build:dates)
  3. Commit the updated indexes & push to gh-pages
